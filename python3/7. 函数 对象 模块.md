#### 函数和过程

> 有返回值的叫函数 没有返回的叫过程, 而python没有过程, 只有函数

```python
#python所有的函数都有返回值 没有return自动为none
def myFunction(*par,exp):
	print("count = " + str(len(par)))
	print("第一个参数:" + par[0])
	return [1,2,3,4]
#return 1,2,3,4 -> tuple 可以不要括号但是需要逗号
print(myFunction("1","2",exp = 4))

```

##### filter 过滤数据

```python
#筛选序列中的满足某个条件的过滤器
#第一个参数是函数
>>> filter(None, (1,2,3,True,False))
<filter object at 0x7f52c790c2b0>
>>> list(filter(None, (1,2,3,True,False)))
[1, 2, 3, True]

>>> list(filter(lambda x: x == 2, (1,2,3,True,False)))
[2]
```
