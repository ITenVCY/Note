> **将某种特殊类型的实例指派给另一个类的属性**

![1567670443788](C:\Users\yx\Desktop\学习笔记\python3\1567670443788.png)

> 特殊类型就是指要实现上述三个方法中的一个

```python
class A: #特殊类
	def __init__(self,aget = None,aset = None,adel = None):
		self.aget = aget
		self.aset = aset
		self.adel = adel

	def __get__(self,instance,owner):
		print("get--self = ",self,"\tins = ",instance,"\tow = ",owner)
		self.aget(instance)		


	def __set__(self,instance,value):
		print("set--self = ",self,"\tins = ",instance,"\tvalue = ",value)
		self.aset(instance,value)


	def __delete__(self,instance):
		print("del--self = ",self,"\tins = ",instance)
		self.adel(instance)


class Test: #
	def __init__(self,x = 0):
		self.x = x

	def getx(self):
		return self.x

	def setx(self,value):
		self.x = value

	def delx(self):
		del self.x

	X = A(getx,setx,delx)




t1 = Test()

t1.X
t1.X = 2
del t1.X


#输出
get--self =  <__main__.A object at 0x7f9e877ec160> 	ins =  <__main__.Test object at 0x7f9e877ec1d0> 	ow =  <class '__main__.Test'>
set--self =  <__main__.A object at 0x7f9e877ec160> 	ins =  <__main__.Test object at 0x7f9e877ec1d0> 	value =  2
del--self =  <__main__.A object at 0x7f9e877ec160> 	ins =  <__main__.Test object at 0x7f9e877ec1d0>
#理解
"""
	用C的角度来看这个过程应该是
	A类   一个拥有三个函数指针的数据成员, 并定义了三个特殊的处理函数,分别对应定义的三个函数指针
	Test类 初始化了对一个数据的操作形式
	
	当X值被获取的时候触发A类中get函数,
	当X值发生改变的时候触发A类中set函数
	删除则触发del
"""
```
